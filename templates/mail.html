Setup Guides
Office 365
(Graph API)
Microsoft SMTP Basic Auth kullanim disi. Graph API kullaniliyor:
1. Azure Portal → Azure Active Directory → App registrations → New registration
2. Application (client) ID ve Directory (tenant) ID'yi kopyala
3. Certificates & secrets → New client secret → degeri kopyala
4. API permissions → Add permission → Microsoft Graph → Application → Mail.Send
5. Grant admin consent tikla
6. From Address: mail gonderecek kullanicinin email adresi (mailbox olmali)
